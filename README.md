# 💣 CCNA Final Project - Enterprise Network

## 📌 Overview
This project simulates an enterprise network using Cisco Packet Tracer with routing, switching, and security configuration.

## ⚙️ Features
- OSPF Dynamic Routing
- VLAN-based switching
- ACL security rules
- DMZ network design
- Enterprise topology simulation

## 🧠 Technologies Used
- Cisco Packet Tracer
- Cisco IOS
- OSPF Routing Protocol
- Access Control Lists (ACL)
- VLANs

## 📁 Project Structure
- configs/ → Router and switch configurations
- labs/ → Theory and explanations
- website/ → Simple dashboard UI
- packet-tracer/ → Network simulation file
- images/ → Topology diagram

## 🚀 How to Run
1. Open `enterprise-network.pkt` in Cisco Packet Tracer
2. Apply configurations from `/configs`
3. Test connectivity between devices
