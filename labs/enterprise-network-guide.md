# 🧠 Enterprise Network Guide

## 🌐 OSPF
OSPF is a dynamic routing protocol used to share routes between routers automatically.

✔ Used in ISP networks  
✔ Fast convergence  
✔ Scalable

---

## 🏢 VLAN
VLAN separates network traffic logically.

✔ Improves security  
✔ Reduces broadcast traffic  

---

## 🛡️ ACL (Access Control List)
ACL controls traffic flow.

In this project:
- Blocks access to internal LAN
- Allows only HTTP/HTTPS to DMZ

---

## 🌍 DMZ (Demilitarized Zone)
A secure zone for public services.

✔ Web server is placed here  
✔ Protects internal network  

---

## 💥 Security Concept
This project simulates:
- ISP backbone routing
- Enterprise segmentation
- Attack protection
